```
给定一个整数数组a，分别求出满足下列条件的子串数量：
1）子串[l, r]中的最大值为a[r]；
2）子串[l, r]中的最小值为a[l]，最大值为a[r]
注意是子串不是子序列，第一问要求O(n)的时间复杂度，第二问要求O(n*lgn)
```
